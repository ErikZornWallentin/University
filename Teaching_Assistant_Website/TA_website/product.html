Azmara
Mobile RPG for Android / iOS


Azmara was inspired from the Zelda series, we grew up playing RPG games like Zelda and we wanted to create a product that was innovative while still had the same "immersion" that Zelda created.

Azmara features several new innovative features such as:

  • 2.5D "faux 3D"
  • Phone / Tablet support
  • Online capabilities
